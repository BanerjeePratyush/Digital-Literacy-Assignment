Project Overview:
This project was completed as part of the Digital Literacy course at VIT. It focuses on developing essential digital skills required for students in today’s rapidly evolving technical world. The project includes a series of practical tasks designed to build a professional digital presence, including creating an educational infographic, building a multi-platform digital portfolio, mastering version control via GitHub, practicing professional email etiquette, and developing a deep understanding of cybercrime awareness in the Indian context.

-------

Project Tasks:
Task 1: Digital Literacy Infographic – Designed a "Digital Literacy and Safety" visual guide using Canva and conducted a peer survey via Google Forms to analyze digital habits and online safety awareness.

Task 2: Digital Portfolio – Established a professional online presence across LinkedIn, GitHub, and Kaggle. This included creating a specialized Profile README that outlines my branch, year of study, and career goals.

Task 3: Platforms Exploration (GitHub Basics) – Mastered the fundamentals of Git and GitHub by creating a structured repository, managing directories, and using version control to document my academic projects.

Task 4: Email Etiquette – Developed professional communication skills by drafting formal emails for internship inquiries and assignment extensions, along with a "Social Media Do’s and Don'ts" checklist.

Task 5: Cybercrime Awareness – Authored a comprehensive case study on UPI and Online Payment Fraud, including a step-by-step prevention guide and reporting protocols for the National Cyber Crime Portal (1930 Helpline).
